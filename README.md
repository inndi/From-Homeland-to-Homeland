# **From Homeland to Homeland**

**From Homeland to Homeland** is a responsive introductory page where you can find interesting information about the homeland of the inhabitants of the town of Practicum.

## Project features

- Semantic HTML5
- Flexbox
- Grid
- Positioning
- Flat BEM

## Technologies used:

* HTML
* CSS
